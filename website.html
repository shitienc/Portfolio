Call Me Handy - Startup Mission

About:

"Runrejsen" was my first project at Zealand, which was about developing a concept for a fictional space travel program. In addition, we had to create a website by using HTML and CSS for the chosen concept. After going through a lot of brainstorming, our group decided to develop the concept: Hotel Alien.

My role:

My role throughout this project was mainly to code the page "Packages & Offers". Moreover, I participated in the development processes, such as the concept-, target group analysis-, and project writing process.

The solution:

The overall project resulted in the creation of a user friendly website for our target audience. Thus, users can easily navigate around the website and obtain the needed information. The website was established by understanding the concept and the target group, which was done throughout out processes.
